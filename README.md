# 自我内在观察

> 探索你的性格密码，发现内在力量

本项目包含两份自评问卷，均为纯前端单页应用（SPA），无需后端服务，直接在浏览器中运行。

---



## 🧩 MBTI 测试

基于荣格心理类型理论与 Myers-Briggs 类型指标（MBTI），通过 **28 道题目** 测定你的四维人格类型。

| 特性 | 说明 |
|------|------|
| 题目数量 | 28 题（每个维度 7 题） |
| 预计耗时 | 约 5 分钟 |
| 结果类型 | 16 种人格类型（INTJ、ENFP 等） |
| 语言支持 | 简体中文 / 繁體中文 / English |
| 主题模式 | 浅色 / 深色 / 跟随系统 |
| 特色功能 | 性别选择 → 角色插图、维度百分比分析、职业/关系/发展建议 |

### 四维人格维度

- **E / I** — 外向 / 内向（Extraversion / Introversion）
- **S / N** — 实感 / 直觉（Sensing / Intuition）
- **T / F** — 思考 / 情感（Thinking / Feeling）
- **J / P** — 判断 / 感知（Judging / Perceiving）

---

## 🌱 自我成长与内在力量评估

通过 **12 道 Likert 5 点评分题目**，评估你在过去六个月中的内在力量状态。

| 特性 | 说明 |
|------|------|
| 题目数量 | 12 题 |
| 预计耗时 | 约 3 分钟 |
| 评分范围 | 12–60 分（分数越高，成长空间越大） |
| 结果等级 | 4 级（稳健独立型 → 探索成长型 → 寻求支持型 → 渴望依托型） |
| 语言支持 | 简体中文 / 繁體中文 / English |
| 主题模式 | 浅色 / 深色 / 跟随系统 |
| 特色功能 | 导航圆点、自动跳题、四维度雷达分析、哲思箴言引用 |

### 四维评估维度

1. **期待与现实** — 自我中心倾向（Egocentric Tendency）
2. **独立与依托** — 依赖倾向（Dependent Tendency）
3. **情绪流动性** — 情绪调节力（Emotional Regulation）
4. **责任内归因** — 担当力（Internal Attribution）
